Easy Git

Easy Git es una guía rápida de comando básicos de git para el manejo de repositorios en proyecto de software. Cada comando tiene una breve descripción y los flag más utilizados.

Git Config

El comando git config obtiene y establece variables de configuración que controlan el aspecto y funcionamiento de Git.

--list

El flag --list o -l en su version abreviada. Enumere todas las variables establecidas en el archivo de configuración, junto con sus valores. 

git config --list
Copy 

git config -l
Copy

Obtener valor de configuración

Para obtener un valor en especifico del archivo de configuracion, debemos utilizar git config seguido del atributo que deseamos obtener. 

git config user.name
Copy 

git config user.email
Copy

Establecer valor de configuración

Para establecer algun valor en especifico del archivo de configuracion, debemos utilizar git config seguido del atibuto y el valor a establecer. 

git config user.name "Tomas Sanders"
Copy 

git config user.email "myemail@mail.com"
Copy

Git Init

El comando git init crea un repositorio Git vacío o reinicialisa uno existente. Este comando crea un repositorio Git vacío, básicamente un directorio .git con subdirectorios para objetos, referencias / cabezas, referencias / etiquetas y archivos de plantilla. También se crea un archivo HEAD inicial que hace referencia al HEAD de la rama maestra. 

git init
